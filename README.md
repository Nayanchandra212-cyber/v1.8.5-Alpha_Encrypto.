# 🛡️ Encrypto v1.8.6 Alpha [OCTA-CORE BUILD]

Encrypto is a Zero-Trust, 100% Local, highly aggressive encryption vault. Built with FOSS principles (No APIs, No Cloud, No Tracking), it utilizes a custom 9-Engine Architecture to provide military-grade steganography, hardware-accelerated UI shape-shifting, and active anti-forensic defenses.

---

## ⚙️ The 9-Engine Framework
Encrypto is powered by a modular, enterprise-grade architecture ensuring zero crashes and absolute security:
* **OME (Orchestration & Maintenance Engine):** The central kernel governing the entire app lifecycle.
* **EPA (Execution & Protection Architecture):** Manages memory flow and acts as an aggressive garbage collector, wiping RAM instantly after decryption.
* **EEA (Evasion & Enchantment Architecture):** The Auto-Heal matrix that actively repairs corrupted data arrays.
* **CME (Cryptographic Mutation Engine):** The mathematical core handling the ciphers and Reverse Salt logic.
* **VDE (Visual Dynamics Engine):** Drives the hardware-accelerated, shape-shifting UI themes.
* **HIE (Haptic Input Engine):** Controls the Incognito Virtual Keyboard and Web Vibration API.
* **SOP (Stealth Override Protocol):** The paranoid surveillance monitor powering the DEFCON Shield and Chameleon mode.
* **ACE (Asynchronous Compute Engine):** Pushes heavy AES/3DES math to background threads to keep the UI running at a buttery-smooth 120Hz.
* **ICE (Isolated Containment Engine):** A quarantine sandbox that sanitizes DOM input to prevent browser-extension injections.

## 🔐 Cryptographic Arsenal
* **Dynamic Multi-Chain Builder:** Route your payload through up to 3 consecutive layers of encryption.
* **Expanded Cipher Suite:** Full support for **AES-256**, **3DES**, **RC4**, **RC4-DROP**, and **Rabbit**.
* **Custom ROT Engine:** Define your own mathematical alphabet shifts before locking it with block ciphers.
* **10-n Reverse Salt Engine:** Mathematical salt length obfuscation hidden dynamically within the ciphertext.
* **Atbash RAM Scrambler:** Bit-flips data strings in volatile memory to defeat forensic RAM scraping tools.
* **Ghost Steganography:** Wraps the final encrypted blob in a fake "system log" sentence to hide it in plain sight.

## 🥷 Stealth & Anti-Surveillance
* **DEFCON 5-Second Shield:** If the browser loses focus (app switch, notification, password manager overlay), a red lockdown screen covers the data. If not recovered in 5 seconds, the system permanently purges the session to `about:blank`.
* **Chameleon Tab:** One-click disguise that instantly swaps the website title and icon to "Google Search" to defeat shoulder-surfing.
* **Incognito Virtual Keyboard:** A built-in, on-screen HTML keyboard that bypasses OS-level keyloggers (like Gboard) while providing physical haptic feedback.
* **Anti-Debugger Trap:** A silent, infinite `debugger;` loop that instantly freezes the browser tab if an attacker attempts to open F12 Developer Tools.

## 🏥 Auto-Recovery & Stability
* **EEA Auto-Heal Matrix:** If a copied vault is missing padding, contains accidental whitespace, or has hidden steganography text attached, the engine intercepts the fatal crash and automatically repairs the math to rescue your data.
* **ICE Sanitization:** Strips invisible HTML tags and tracking pixels sometimes injected by browser extensions (like Grammarly) before encryption.
* **Zero-API Dependency:** 100% offline functionality. It works entirely in your local browser cache.
