# Encrypto v1.8.5 Alpha

Encrypto is a zero-knowledge, local-first encryption suite designed for high-privacy mobile and desktop use. 

## 🚀 Features
- **Shape-Shifter UI:** 3 distinct UI layouts (Hacker, Modern, Lite) that physically rearrange elements on the screen.
- **Stealth Chameleon:** Instantly mask your browser tab to look like a generic Google search.
- **Haptic Virtual Keyboard:** Built-in keyboard that prevents system-level keylogging and provides physical vibration feedback.
- **Emergency Purge:** Automatic session wipe if you switch apps or click the purge button.

## 🛠️ How to use
1. **Select Theme:** Choose the UI that fits your environment.
2. **Select Cipher:** AES-256 is recommended for standard security.
3. **Input Data:** Use the virtual keyboard to enter your secret message and passphrase.
4. **Encrypt:** Save the output blob.

## ⚠️ Important
This app operates 100% locally. If you lose your passphrase or forget your Custom ROT settings, your data is mathematically unrecoverable.
