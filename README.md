
# Encrypto v1.8.5 Alpha [FORTRESS BUILD]

**Original Project By:** Omkar Das  
**Upgraded & Hardened By:** Nayan Chandra  

Encrypto is an aggressive, Zero-Trust, 100% local encryption vault. Built entirely with FOSS principles (No APIs, No Cloud, No Tracking), it utilizes hardware-accelerated themes and the Web Vibration API for a physical, system-level security experience.

## 🛡️ The Fortress Upgrades
- **Multi-Chain Builder:** Route your data through up to 3 consecutive layers of encryption (e.g., AES-256 ➔ RC4 ➔ Custom ROT).
- **Custom ROT-N Engine:** Define your own mathematical alphabet shifts before applying block ciphers.
- **Haptic Virtual Keyboard:** A built-in, bottom-tray HTML keyboard that bypasses OS-level keyloggers (like Gboard) and provides physical vibration feedback on supported Android devices.
- **Dynamic Warning System:** Actively monitors your cipher combinations and alerts you if you apply dangerous or unrecoverable custom layers.
- **Chameleon Stealth Mode:** Instantly masks the browser tab to look like standard system dashboards or school notes to deter shoulder-surfing.
- **Hardware-Accelerated UI:** 4 unique themes (Modern Glass, Hacker Terminal, Lite Fast, Standard) with smooth, CPU-optimized rendering.

## ⚠️ Critical Warning
If you utilize **Multiple Layers** or a **Custom ROT Shift**, you **MUST** memorize your exact configuration. Because Encrypto is 100% local and Zero-Knowledge, there is no password reset. Lost configurations mean mathematically unrecoverable data.

## 📜 Open Source
This project uses the `crypto-js` library for client-side cryptographic operations. No data ever leaves your browser.
